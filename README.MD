# CardMerge
A Balatro mod entire based around manipulating and merging the suits of your playing cards. A flush lover's delight!

![cmerge_splash_screen](cmerge_splash_screen.png)

> [!IMPORTANT]
> As of 11/28/25, the mod is in an alpha state and has not been fully tested with other mods. Unexpected results can and will happen when this mod is played with modded suits.

# Requirements
- Requires `Steamodded >= 1.0.0~BETA-1016c` - [link](https://github.com/Steamodded/smods/wiki).
- Requires `Lovely >= 0.6.0` - [link](https://github.com/ethangreen-dev/lovely-injector). This should already be installed as part of steamodded's installation process.

# Installation
- Follow the instructions on the [Steamodded repository](https://github.com/Steamodded/smods?tab=readme-ov-file#how-to-install-a-mod).
Latest release should be located in this repository's release page.

# Features
The mod comes with 12 new suits, which result from merging the vanilla suits in different ways. Cards with merged suits will have the same logic as a wild card, or if the Smeared Joker is active.
| **Name**  | **Pattern**                                |
|-----------|-------------                               |
| Nones     | A card has no suits. Acts as a fifth suit. |
| Kites     | Diamonds + Spades                          |
| Suns      | Diamonds + Clubs                           |
| Cups      | Hearts + Spades                            |
| Crescents | Hearts + Clubs                             |
| Squids    | Clubs + Spades                             |
| Metals    | Hearts + Diamonds                          |
| Waters    | Hearts + Clubs + Spades                    |
| Earths    | Clubs + Diamonds + Spades                  |
| Fires     | Diamonds + Hearts + Spades                 |
| Airs      | Clubs + Diamonds + Hearts                  |
| Omnis     | Clubs + Diamonds + Hearts + Spades         |

More generalized suits will come in the future for combinations involving modded suits.
